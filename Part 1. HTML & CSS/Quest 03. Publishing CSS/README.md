# Quest 03. Publishing CSS


## Introduction
* 이번 퀘스트에서는 CSS를 이용해 실제 웹 페이지를 모사해 퍼블리싱하는 것에 도전해볼 예정입니다.

## Topics
* 퍼블리싱을 위해 몇 가지 중요한 속성들
  * `font-*`
  * `text-*`
  * `box-sizing`
  * `:hover`/`:active`

## Checklist
* CSS 퍼블리싱을 할 때, class와 selector들은 어떤 식으로 정리하는 것이 좋을까요?

  * CSS 방법론등을 참고하여 팀에게 맞는 정리법을 만드는게 좋다고 생각합니다.
  * 자식, 자손 선택자를 이용하는 것 보다, 그 컨텐츠들에 class를 붙여서 사용하는 것이 더 효율적입니다.
